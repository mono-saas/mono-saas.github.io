# MonoSaaS.com

This is the public repository for the [MonoSaaS.com](https://monosaas.com) landing page.

**MonoSaaS** is the Majestic Monolith starter kit for Next.js. It helps developers self-host their SaaS on a VPS using Docker, avoiding the "Serverless Pricing Cliff."

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- GitHub Pages (Hosting)
